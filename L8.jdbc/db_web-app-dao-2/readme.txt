найскладніший приклад, в якому використовуються джойни

визначено два endpoint-и:
  cars (може приймати параметр acc=firstName, наприклад: cars?acc=admin)
  показує список всіх наявних автомобілів або автомобілів, що належать вказаному користувачу

  accounts (може приймати параметр cars, наприклад: accounts?cars=)
  якщо вказано параметр cars (значення неважливе, перевіряється лише наявність параметра), то для
  користувачів також будуть виводитись автомобілі, що їм належать; в іншому випадку - список
  автомобілів не виводиться, але користувач підсвічується іншим кольором, якщо він володіє
  хоча б одним автомобілем

для повноцінної роботи цього прикладу в базу має бути додана таблиця account_link (файл car_account_link.sql);
спочатку ця таблиця не включена до бази даних car_portal;
