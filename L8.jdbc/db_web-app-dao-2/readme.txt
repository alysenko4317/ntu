самый сложный пример, в котором используются джойны

определено два endpoint-a
  cars (может принимать параметр acc=firstName, например: cars?acc=admin)
  показывает список всех имеющихся машин либо машин, которые принадлежат указанному пользователю

  accounts (может принимать параметр cars, например: accounts?cars=)
  если указан параметр cars (значение неважно, проверяется только наличие параметра) то для пользователей также будут
  выводится принадлежащие им машины; в ином случае - список машин не выводится, но пользователь подсвечивается другим
  цветом, если он владеет хотябы одной машиной 

для полноценной работы этого примера в базу должна быть добавлена таблица account_link (файл car_account_link.sql);
изначально эта таблица не включена в базу данных car_portal;
